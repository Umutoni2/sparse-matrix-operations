# sparse-matrix-operations
## Overview

This project loads two sparse matrices from a file and performs basic operations:

* Addition
* Subtraction
* Multiplication

It is designed to handle matrices efficiently by storing only non-zero values.

## Project Structure

```
/sparse_matrix/
│── code/sparse-matrix.js        # Source code
│── sample-input/          # Sample input files
```

## Features

* Efficient sparse matrix representation
* Supports large matrices
* Performs 3 core operations

## How to Run

1. Navigate to the source folder:

   ```
   cd path
   ```
2. Run the program:

   ```
   node sparse-matrix.js
   ```

## Input

* Matrices are read from a file in the `sample-input/` folder
* Only non-zero values are stored

## Output

* Displays results of:

  * Matrix Addition
  * Matrix Subtraction
  * Matrix Multiplication

## Tech Used

* JavaScript (Node.js)
